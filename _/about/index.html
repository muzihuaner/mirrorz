Usage
  • The following urls are all valid.
  • https://mirrorz.org/_/
  • https://mirrorz.org/_/os/ArchLinux
  • https://mirrorz.org/_/app/Git
  • https://mirrorz.org/_/font
  • https://mirrorz.org/_/list
  • https://mirrorz.org/_/list/pypi
  • https://mirrorz.org/_/site
  • https://mirrorz.org/_/site/BFSU
  • https://mirrorz.org/_/about
  • Use the below script for speed test (requires python3 and curl)!
  • https://mirrorz.org/oh-my-mirrorz.py
  • curl https://mirrorz.org/oh-my-mirrorz.py | python3
  • Experimental Feature: 302 Backend
  • https://mirrors.mirrorz.org/archlinux
  • https://m.mirrorz.org/centos
  • Experimental Feature: Search Backend
  • https://search.mirrorz.org/archlinux/
  • https://s.mirrorz.org/openwrt/snapshots/targets/zynq/generic/sha256sums
